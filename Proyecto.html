Información

Como funciona la Red Lan

Una Red LAN (local Area Network) es un tipo de red informática que conecta dispositivos dentro de un área geográfica limitada, como una casa, una oficina. Esto puede usarse para compartir archivos y datos, Compartir entre dispositivos, Acceso a internet, entre muchas cosas más

Ventajas y Desventajas

Ventajas

  • Alta velocidad
  • Control de datos
  • Compartir datos facilmente

Desventajas

  • Alcance limitado
  • Puede requerir configuración tecnica
  • Vulnerabilidad
